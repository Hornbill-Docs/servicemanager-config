# Supplier Contract Sub-statuses

Contracts come with three statuses. Draft, Active, and Archived. For each of these statuses you can define custom sub-statues that allow you to define additional states that a contract might be in.  

## Where sub-statuses are used

A sub-status can be selected when editing a Supplier Contract record. Once added, the sub-status will be visible in the following areas:

* **Supplier Contracts list**: When the list of contracts is being viewed, a column that displays the sub-status for each contract is available.  If this column is not visible, it can be added using the list configuration.
