# HAi Assistant Settings

The HAi Assistant Settings allow you to configure the behavior of the HAi Agent and User Assistants, including guardrails, customization, and data access rules.


## Configuration

The configuration options allow you to select which HAi features are enabled.

* **HAi Agent Assistant**: Enable or disable the HAi Agent Assistant.
* **HAi User Assistant**: Enable or disable the HAi User Assistant.
* **HAi Search**: Enable or disable the HAi Search feature.
* **HAi Insights**: Enable or disable the HAi Insights feature.

## HAi Agent Assistant

The HAi Agent Assistant is a virtual assistant that can be used to help agents with their work. The assistant can be configured to provide information, answer questions, and perform tasks on behalf of the agent.

### Agent Guardrails

Guardrails are rules that can be applied to the HAi Agent Assistant to restrict its behavior. Guardrails can be used to prevent the assistant from performing certain actions, or to limit the information that it can access.

### Agent Customization

The customization options allow you to configure the behavior of the HAi Agent Assistant. You can customize the tone of the assistant's responses.

## HAi User Assistant

The HAi User Assistant is a virtual assistant that can be used to help support users to find solutions or answers. The assistant can be configured to provide information, answer questions, and perform tasks on behalf of the user.

### User Guardrails

User Guardrails are rules that can be applied to the HAi User Assistant to restrict its behavior. Guardrails can be used to prevent the assistant from performing certain actions, or to limit the information that it can access.

### User Customization

The customization options allow you to configure the behavior of the HAi User Assistant. You can customize the tone of the assistant's responses.
