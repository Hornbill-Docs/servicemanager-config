# Text Assist

HAi Text Assist is a generative AI-powered tool within Hornbill Service Manager designed to boost agent productivity by helping users instantly create, edit, and format text anywhere they post comments or draft emails. It eliminates the need to start from scratch or manually edit content.

Key features and actions include:

* **Ask HAi**: Generates step-by-step instructions or solutions to common issues using simple prompts.
* **Change Tone**: Automatically adjusts the tone of your text to suit different audiences with four different options.
* **Improve Text**: Fixes grammar and spelling mistakes and clarifies ambiguous statements.
* **Shorter Text**: Compresses long text content without losing the original meaning.
* **Listify**: Transforms large, dense blocks of text into readable, step-by-step lists.
